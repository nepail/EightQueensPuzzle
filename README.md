# 8 皇后問題

這個專案是一個用來解決八皇后問題的 Node.js 應用程式。

## 簡介

八皇后問題是一個經典的棋盤問題，旨在找出如何在 8×8 棋盤上放置 8 個皇后，使得任何兩個皇后都不互相攻擊。

## 使用方式

1. **克隆專案：**
   ```bash
   git clone https://github.com/nepail/EightQueensPuzzle.git


2. **安裝依賴**
   ```bash
   cd 8-queen-puzzle
   npm install

3. **執行程式**
   ```bash
   node index.js
